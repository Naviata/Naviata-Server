# Naviata-Server
All necessary information and files for hosting Naviata on your own DayZ Standalone server.

# Server Installation
**Basics**

* Make sure you have a up-to-date *@Naviata* folder in your server root folder.
* Copy the Naviata.bikey from *Naviata_Server_Main/Keys/* folder.
* Get the mission of choice alongside default server config from Extras folder in the Naviata Experience mod.

**Advanced setup**

If you are running a dedicated machine with multiple DayZ servers, it is recommended to subscribe to our custom server packages for Naviata, More information will be added here soon as we work on making these available to the public.

# FAQ: Frequently Asked Questions

**Q: Can I repack Naviata into my other mod?**

A: NO, You can although use Steam's Workshop collection feature!

**Q: Is Naviata compatible with other mods?**

A: I do not have any comprehensive list of what works and what not. The Terrain has been made as light/basic as possible and would be compatible with plenty of mods out there. As can be seen hosted by other Server Providers.

**Q: I want to host only Naviata. Which mission should I use?**

A: Use the *Default* or *Hardcore* mission and remove the include of *types_dzn.xml* from *cfgeconomycore.xml*. 
Doing this will stop spawning custom items from the *Naviata Experience* package.

**Q: Where can I find a default mission for Naviata?**

A: Look into the Mission Files folder in this repository, Naviata as of now has two missions available; Default and Hardcore.

**Q: Where can I find the default CE Tools project files for Naviata?**

A: Look into the *Central Economy Tool* folder in this repository. Naviata is using area territories for most of the animals (rather than territories per species). As for areaflags.map, that is used only for the tier definition.
